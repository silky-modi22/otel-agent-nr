Telemetry Playground

Connecting…

Generate, transform, and export OpenTelemetry data without wiring a full stack first. Follow the four steps below in order.

How to run & setup guide
1

One-time setup — from the repo root: pip install -r requirements.txt then ./scripts/build-collector.sh

2

Start the server — python -m agent serve --http-port 8000

3

Open this page at http://127.0.0.1:8000/ — it connects automatically.

4

Step 1 — Start the collector. Pick where telemetry goes: ClickHouse only, New Relic + ClickHouse, New Relic only, or Local debug (prints to console, exports nothing). The collector listens on :4318 and must be running before you start the agent.

5

Step 2 — Start the agent. Choose what generates telemetry: synthetic sample traces, an AI-shaped sample payload, fetch GitHub events once → ClickHouse, or stream GitHub events continuously → ClickHouse. The two GitHub buttons auto-start the ClickHouse collector if it isn't running yet.

6

Step 3 — Verify. Refresh the ClickHouse tab to see rows land in otel.otel_traces, or the New Relic tab for NR modes. Use Step 4 to stop the agent, the GitHub stream, or everything at once.
